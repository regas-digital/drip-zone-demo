# Drip Zone - Démo E-commerce Streetwear (Haïti)

Ceci est une démo de site e-commerce de streetwear, nommée **Drip Zone**, conçue pour le marché haïtien. Elle présente une page d'accueil moderne et un aperçu de l'espace administrateur.

## Contenu de la Démo

| Fichier | Description |
| :--- | :--- |
| `index.html` | Page d'accueil de la boutique en ligne. |
| `admin.html` | Simulation de l'espace administrateur (Dashboard). |
| `style.css` | Fichier de style (CSS) pour les deux pages. |
| `assets/` | Dossier contenant les images des produits. |

## Comment visualiser la démo

1.  **Page d'accueil :** Ouvrez le fichier `index.html` dans votre navigateur.
2.  **Espace Admin :** Ouvrez le fichier `admin.html` dans votre navigateur.

Le lien vers l'espace admin est également présent dans le menu de navigation de la page d'accueil (icône d'engrenage).
